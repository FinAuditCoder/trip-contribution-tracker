# trip-contribution-tracker
Overview: This project is an Excel-based financial contribution tracking system designed to monitor group savings, individual payments, and overall collection progress. It demonstrates structured financial data management, automated calculations, and reporting using Pivot Tables.
Objectives: Track monthly contributions for a group of 10 mwmbers, monitor individual payment progress, automatically calculate remaining balances, generate monthly collection summaries and provide structured reporting for financial oversight.
Features Implemented: Structured Raw Data table, automated calculations using IF logic, balance monitoring system, status automation (Fully Paid/Owing), pivot table monthly summary, individual contribution reporting, currency formatted in ZAR (R) and frozen header panels for usability.
Tools used: Microsoft excel, pivot tables, logical functions and basic financial control structure.
Why this project matters? This project reflects foundational financial systems thinking aligned with: Internal Audit, Risk & Compliance monitoring, Financial data structuring as well as Reporting and oversight processes. It represents the first step in transitioning from finance operations into technology-driven financial systems.
