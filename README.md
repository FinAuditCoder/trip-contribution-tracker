# trip-contribution-tracker
## Overview
This project is an Excel-based financial contribution tracking system designed to monitor group savings, individual payments, and overall collection progress. It demonstrates structured financial data management, automated calculations, and reporting using Pivot Tables.
## Objectives 
-Track monthly contributions for a group of 10 mwmbers
-Monitor individual payment progress
-Automatically calculate remaining balances
-Generate monthly collection summaries
-Provide structured reporting for financial oversight
## Features Implemented
-Structured Raw Data table
-Automated calculations using IF logic
-Balance monitoring system
-Status automation (Fully Paid/Owing)
-Pivot table monthly summary
-Individual contribution reporting
-Currency formatted in ZAR (R)
-Frozen header panels for usability
## Tools used 
-Microsoft excel
-Pivot tables
-Logical functions
-Basic financial control structure
## Why this project matters
This project reflects foundational financial systems thinking aligned with: Internal Audit, Risk & Compliance monitoring, Financial data structuring as well as Reporting and oversight processes. It represents the first step in transitioning from finance operations into technology-driven financial systems.
